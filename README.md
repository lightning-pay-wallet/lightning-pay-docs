## What is Lightning Pay Wallet

Lightning Pay Wallet is a non-custodial Bitcoin and Lightning Network wallet: the app generates a private key on your own device (encoded as a 12-word BIP-39 seed phrase) rather than creating an account on a company server, so the operator never holds or controls user funds.

Alongside standard on-chain and Lightning BTC balances, the wallet holds two dollar-pegged stablecoins (USDC and USDT) at the same seed, and links to a Bitcoin Visa card that can spend directly from any of those balances at the point of sale, with no monthly fee and 0% FX markup.

The wallet is free to install (Windows, macOS, Linux), with no account required and no KYC to use the app itself — KYC only applies where the buy/sell partner rails (MoonPay, Strike) are legally required to collect it.

## About

Lightning Pay Wallet is developed and maintained as a non-custodial Bitcoin and Lightning wallet with an integrated Visa card, stablecoin support, bill pay, automated recurring purchases, and cost-basis tax tracking. This repository hosts reference documentation for the product; the wallet itself is downloaded from [lightningpay.info/download](https://lightningpay.info/download).

This is informational documentation, not financial, legal, or tax advice.
